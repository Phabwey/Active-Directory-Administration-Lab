# User and Group Management

## Overview

User and group management is a core responsibility of system administrators. This lab demonstrates the creation and management of user accounts, security groups, and permissions in an Active Directory environment.

---

## Creating User Accounts

The following user accounts were created:

| Username | Department | Role |
|----------|------------|--------|
| paballo.litabe | IT Support | Technician |
| john.smith | Finance | Accountant |
| sarah.jones | Human Resources | HR Manager |
| david.brown | Sales | Sales Representative |

---

## User Account Configuration

The following settings were configured:

- Username and display name
- Password requirements
- Department information
- Job title
- User logon permissions
- Account status

---

## Creating Organizational Units (OUs)

The following Organizational Units were created:

- IT Support
- Finance
- Human Resources
- Sales
- Management

These OUs help organize users and simplify administration.

---

## Security Groups

The following security groups were created:

| Group Name | Purpose |
|------------|----------|
| IT-Support | Manage technical systems |
| Finance-Team | Access financial resources |
| HR-Team | Manage employee records |
| Sales-Team | Access sales resources |

---

## Group Membership

Users were assigned to the appropriate groups based on their department and responsibilities.

Example:

- paballo.litabe → IT-Support
- john.smith → Finance-Team
- sarah.jones → HR-Team
- david.brown → Sales-Team

---

## Administrative Tasks Performed

- Created user accounts.
- Modified user properties.
- Enabled and disabled accounts.
- Reset passwords.
- Assigned users to groups.
- Removed users from groups.
- Updated account permissions.
