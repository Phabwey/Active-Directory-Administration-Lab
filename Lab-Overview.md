# Active Directory Lab Overview

## Introduction

This lab simulates a corporate Active Directory environment and demonstrates essential administrative tasks performed by IT Support Specialists and Junior System Administrators.

The project focuses on user management, security policies, organizational units, group policies, and troubleshooting within Windows Server environments.

---

## Lab Objectives

- Configure Active Directory Domain Services (AD DS).
- Create and manage user accounts.
- Organize users into Organizational Units (OUs).
- Create and manage security groups.
- Configure password and account lockout policies.
- Apply Group Policy Objects (GPOs).
- Troubleshoot authentication and access issues.

---

## Lab Components

### Domain Controller

The Domain Controller manages authentication, authorization, and centralized administration of users and devices within the domain.

### Organizational Units (OUs)

Organizational Units are used to organize users, computers, and groups according to departments and business requirements.

Example structure:

- Management
- Human Resources
- Finance
- IT Support
- Sales

### Security Groups

Security groups simplify permission management by assigning access rights to multiple users simultaneously.

Examples:

- Domain Admins
- IT Support
- Finance Team
- HR Team
